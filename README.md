## Infos zum Projekt
Am Besten den Frontend und Backend Ordner getrennt öffnen um den Code zu runnen. Im Frontend `npm run dev` eingeben um den Server zu starten.

Die API Calls werden im Frontend gemacht, um die Laufzeit zu reduzieren und da es sich um nicht sensitive Daten handelt.
